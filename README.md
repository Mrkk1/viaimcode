# ViaimCode

使用 AI 生成网页的智能工具 | AI-powered Website Generator

## 功能特点

- 🤖 AI 驱动的网页生成
- 💻 实时预览和编辑
- 📱 响应式设计支持
- 🎨 现代化UI界面
- 🔄 代码实时同步
- 📤 一键导出和分享

## 技术栈

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Shadcn/ui

## 开发环境设置

1. 克隆仓库：
```bash
git clone https://codeup.aliyun.com/621dc90cb49717e05aee119c/AI/viaimcode.git
```

2. 安装依赖：
```bash
cd viaimcode
npm install
```

3. 配置环境变量：
复制 `.env.example` 到 `.env.local` 并填写必要的配置。

4. 启动开发服务器：
```bash
npm run dev
```

## 环境要求

- Node.js 18+
- npm 或 yarn
